# Lung Cancer Classification using Machine Learning

## Overview

This project is a machine learning-based system for classifying lung cancer CT scan images as normal or abnormal. The application compares the performance of different machine learning algorithms including Support Vector Machine (SVM), K-Means Clustering, and Logistic Regression.

The system also includes a GUI interface built using Tkinter for dataset upload, model execution, prediction, and accuracy visualization.

## Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn
* Matplotlib
* Tkinter
* Pandas

## Algorithms Implemented

* Support Vector Machine (SVM)
* K-Means Clustering
* Logistic Regression
* PCA (Principal Component Analysis)

## Features

* CT scan image preprocessing and normalization
* Dataset train-test splitting
* PCA-based dimensionality reduction
* Machine learning model training and evaluation
* Accuracy comparison graph
* GUI-based lung cancer prediction system
* Prediction of normal vs abnormal CT scans

## Project Workflow

1. Upload CT scan dataset
2. Read and preprocess image data
3. Apply PCA dimensionality reduction
4. Train machine learning models
5. Compare model accuracies
6. Predict lung cancer using uploaded CT scan images

## Files Included

* `SVM_KMEANS.py` → Main GUI application
* `test.py` → Model training and testing script
